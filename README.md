# QD-Assignment
The user is authenticated and authorized for adding cars along with specifications to the list. Passport JS has been used for authentication.

# Technology
    1. Node.js
    2. Express
    3. MongoDB
    4. Mongoose
    5. EJS

## Setup
   Make sure that Node.js, Mongodb, VS Code(any text editor) and  are installed in your local system. 
   Open the terminal and type the following command
   * npm install (Tod download all the dependencies)
   * To run the project type **npm start** in the terminal

## Functionality
The functionalities that are implemented in this project include
    
    1. User Sign Up, Sign In and Sign Out
    2. Add cars to the list.
